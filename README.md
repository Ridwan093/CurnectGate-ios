# CurnectGate+  (Andriod and IOS)

## Team Direction Guide

>  CurnectGate+: A comprehensive mobile and web-based gated community management system designed to streamline visitor access, facilitate communication, manage payments, and enhance security.
> 

## `Tech Stack`

**Framework:** Flutter


## Clone Project

Clone the project to your local machine

```bash
  git clone https://github.com/Boacon-Synergy-Inc/curnectGate-mobileApp.git
```


## Build Android:

```bash
  flutter build apk --target-platform --split-per-abi
```

## Build IOS:

```bash
  flutter build ios
```

## Build Windows:

```bash
  flutter build linux
```

## Build web:

```bash
   flutter build web
```
